# 🤖 Auto Reply Chatbot (Python)

An automated chatbot built using Python that replies to messages instantly using predefined logic. This project focuses on reducing repetitive typing and making communication faster and more efficient.

---

## 🚀 Features

* 💬 Automatic message replies
* ⚡ Fast and real-time response
* 🧠 Simple and customizable logic
* 🔁 Reduces manual effort
* 🛠 Beginner-friendly project

---

## 📌 Use Cases

* Chat automation
* Quick reply system
* Learning Python automation
* Basic chatbot development

---

## 🛠 Tech Stack

* Python
* pyautogui
* pyperclip
* time

---

## ⚙️ How It Works

1. Detects incoming message
2. Copies the message
3. Processes text using conditions
4. Matches with predefined replies
5. Sends response automatically

---

## ▶️ Installation

```bash
git clone https://github.com/your-username/auto-reply-chatbot.git
cd auto-reply-chatbot
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python 03_bot.py
```

---

## 🔧 Customization

Edit reply logic in the code:

```python
if "hello" in message:
    reply = "Hi there!"
elif "bye" in message:
    reply = "Goodbye!"
```
